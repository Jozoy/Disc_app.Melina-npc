# Melina Bot

Melina Bot is a simple, fun, and interactive Discord bot designed for friends, inspired by the NPC **Melina** from *Elden Ring*. As a huge *Souls* fan, this bot is here to enhance your Discord experience with helpful responses, jokes, and more!

## Features
- Responds with random, lore-inspired quotes and messages
- Customizable responses
- Fun interactions, inspired by the world of Elden Ring
- A helpful companion for your Discord server
- Private messaging support using '?' prefix

## Prerequisites
- Python 3.8 or higher
- A Discord account and bot token
- Git (for cloning the repository)

## Installation
1. Clone this repository:
```bash
git clone <your-repository-url>
cd DISCORD_Bot-JAVONA
```

2. Install the required dependencies:
```bash
pip install -r requirements.txt
```

3. Create a `.env` file in the root directory and add your Discord bot token:
```
DISCORD_TOKEN=your_token_here
```

## Required Packages
- discord.py
- python-dotenv
- aiohttp
- requests

## Running the Bot
Start the bot:
```bash
python Melina_bot.py
```

## Usage
- Send normal messages in any channel to interact with the bot
- Prefix your message with '?' for private responses
- The bot will respond based on the message content
